# Red-Things
IOT(Internet of things) platform based on Node-Red, Easy but very useful to built a iot system

# init project
> npm install

# start to run
> npm run server

# interface
+ /login  all url's entry
+ /main   the main dashboard
+ /red    node-red editor
+ /api    node-red http interface
